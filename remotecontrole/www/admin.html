  • Quels sont les outils nécessaires pour procéder à ces modifications?
    Il vous est fortement conseillé de posséder un éditeur de code (comme par exemple Sublime Text) par soucis de lisibilité de ce que vous allez éditer.
  • Sur quoi dois-je procéder la modification?
    La modification doit s'effectuer directement dans le projet en lui même, plus précisement dans le fichier data.xml se trouvant dans le fichier "www"
  • Je n'ai pas le projet sur mon ordinateur actuellement, où puis-je le récuperer?
    Si vous n'avez pas encore récupéré le projet, ou si vous l'avez égaré sur votre disque dur, tout son contenu est récupérable directement sur notre github à l'adresse suivante: LIEN
